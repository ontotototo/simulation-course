## ЛР 6-1: Дискретная случайная величина

### Цель работы

Реализовать алгоритм генерации дискретной случайной величины, заданной рядом распределения. Провести серию экспериментов при различных объёмах выборки N = 10, 100, 1 000, 10 000, вычислить эмпирические характеристики и проверить соответствие теоретическому распределению по критерию хи-квадрат.

---

### Теоретические сведения

Дискретная случайная величина X задана рядом распределения:

| X | 1 | 2 | 3 | 4 | 5 |
|---|---|---|---|---|---|
| P | 0.1 | 0.2 | 0.4 | 0.2 | 0.1 |

**Теоретическое математическое ожидание:**

$$M[X] = \sum_{i} x_i \cdot p_i$$

**Теоретическая дисперсия:**

$$D[X] = \sum_{i} x_i^2 \cdot p_i - M^2[X]$$

---

### Алгоритм генерации (метод обратной функции / кумулятивных вероятностей)

1. Вычислить кумулятивные вероятности: $F_k = \sum_{i=1}^{k} p_i$
2. Сгенерировать равномерное случайное число $r \sim U(0, 1)$
3. Найти наименьший индекс $k$, при котором $F_k \geq r$
4. Вернуть соответствующее значение $x_k$

**Реализация в коде:**
```python
cum_probs = np.cumsum(probs)
samples = values[np.searchsorted(cum_probs, np.random.rand(N))]
```

---

### Вычисляемые характеристики

**Выборочное среднее:**
$$\hat{M} = \frac{1}{N} \sum_{j=1}^{N} x_j$$

**Выборочная дисперсия:**
$$\hat{D} = \frac{1}{N} \sum_{j=1}^{N} (x_j - \hat{M})^2$$

**Относительная погрешность:**
$$\delta_M = \frac{|\hat{M} - M|}{|M|} \cdot 100\%$$

---

### Критерий согласия хи-квадрат

Статистика критерия:

$$\chi^2 = \sum_{i=1}^{k} \frac{(n_i - N \cdot p_i)^2}{N \cdot p_i}$$

где $n_i$ — наблюдаемая частота значения $x_i$, $N \cdot p_i$ — ожидаемая частота.

Критическое значение при уровне значимости $\alpha = 0.05$ и числе степеней свободы $df = k - 1 = 4$:

$$\chi^2_{\text{кр}} = \chi^2_{0.95,\, 4} \approx 9.49$$

**Гипотеза:** $H_0$ — выборка подчиняется заданному закону распределения.

- Если $\chi^2 < \chi^2_{\text{кр}}$ → критерий **пройден** (гипотеза не отвергается)
- Если $\chi^2 \geq \chi^2_{\text{кр}}$ → критерий **отклонён** (гипотеза отвергается)

### Вывод по ЛР 6-1

С ростом объёма выборки N:
- Относительные погрешности математического ожидания и дисперсии **монотонно убывают**, что соответствует закону больших чисел.
- При N = 10 погрешности могут быть значительными (десятки процентов), а критерий хи-квадрат может быть отклонён из-за малой выборки.
- При N ≥ 1 000 эмпирические характеристики устойчиво сходятся к теоретическим значениям, критерий согласия хи-квадрат стабильно проходится.
- Метод кумулятивных вероятностей обеспечивает корректную генерацию дискретной СВ.

## ЛР 6-2: Нормальная случайная величина

### Цель работы

Смоделировать нормальную случайную величину $X \sim N(\mu, \sigma^2)$ методом Бокса–Мюллера, построить гистограмму, оценить точность моделирования при N = 10, 100, 1 000, 10 000 и применить критерий хи-квадрат.

---

### Теоретические сведения

Нормальное распределение $N(\mu, \sigma^2)$:

$$f(x) = \frac{1}{\sigma \sqrt{2\pi}} \exp\left(-\frac{(x-\mu)^2}{2\sigma^2}\right)$$

Параметры по умолчанию: $\mu = 0$, $\sigma = 1$.

---

### Метод Бокса–Мюллера

Метод позволяет получать нормально распределённые числа из пары равномерных $U_1, U_2 \sim U(0, 1)$:

$$Z = \sqrt{-2 \ln U_1} \cdot \cos(2\pi U_2)$$

Затем выполняется масштабирование:

$$X = Z \cdot \sigma + \mu$$

**Реализация в коде:**
```python
u1, u2 = np.random.rand(N), np.random.rand(N)
samples = np.sqrt(-2 * np.log(u1 + 1e-10)) * np.cos(2 * np.pi * u2) * sigma + mu
```

> Добавление `1e-10` предотвращает вычисление `log(0)`.

---

### Критерий хи-квадрат для непрерывного распределения

Данные разбиваются на **10 равных интервалов**. Для каждого интервала $[a_i, b_i]$ вычисляется теоретическая вероятность:

$$p_i = F(b_i) - F(a_i) = \Phi\left(\frac{b_i - \mu}{\sigma}\right) - \Phi\left(\frac{a_i - \mu}{\sigma}\right)$$

Ожидаемые частоты нормируются к сумме наблюдаемых:

$$e_i = \frac{p_i}{\sum p_j} \cdot N$$

Статистика:

$$\chi^2 = \sum_{i=1}^{10} \frac{(n_i - e_i)^2}{e_i}$$

Критическое значение при $df = 9$, $\alpha = 0.05$:

$$\chi^2_{\text{кр}} = \chi^2_{0.95,\, 9} \approx 16.92$$

### Вывод по ЛР 6-2

- Метод Бокса–Мюллера корректно воспроизводит нормальное распределение: гистограмма при N ≥ 1 000 хорошо аппроксимирует теоретическую плотность $f(x)$.
- При малых N (10, 100) наблюдаются значительные отклонения эмпирических характеристик от теоретических; критерий хи-квадрат может не выполняться.
- С увеличением N погрешности сходятся к нулю, а критерий согласия стабильно подтверждает нормальность распределения.
- Гистограмма при N = 10 000 практически совпадает с теоретической кривой нормального распределения.
