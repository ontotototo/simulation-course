### Теория 1
Вероятность события можно представить геометрически как отрезок на интервале [0, 1]:

$$P\{\text{random} < p\} = p$$

где random — случайная величина, равномерно распределённая на [0, 1], а p — вероятность наступления события.

В коде используется следующий алгоритм:

1. A := α (генерируем случайную величину)
2. k := 1
3. A := A - pₖ
4. Если A ≤ 0 → произошло событие Aₖ
5. Иначе k := k + 1, повторить шаг 3


### Теория 2
Полная группа событий

События $A_1, A_2, ..., A_m$ образуют полную группу, если:

$$\sum_{i=1}^{m} p_i = 1$$

где $p_i$ = P{Aᵢ} — вероятность наступления события $A_i$.
